# Nowaste
O Nowaste é um sistema orientado a objetos feito para empresas, como supermercados, que precisam controlar a validade dos produtos e evitar desperdícios. Ele permite cadastrar e gerenciar os itens em estoque, organizando tudo de forma estruturada com classes e objetos.
O sistema alerta quando produtos estão prestes a vencer, mostra quais vão vencer no mês e também permite identificar quais podem ser trocados com o fornecedor.
Com isso, a empresa consegue ter mais controle sobre o estoque, reduzir perdas e tomar decisões mais rápidas sobre os produtos.
